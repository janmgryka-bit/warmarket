# War Market

War Market is a Godot 4 prototype for a stylized historical auto-battler.

## Project Structure

- `godot_project/war-market/` - Godot runtime project, scripts, scenes, and imported game-ready assets.
- `design/` - design notes, art direction, asset roadmap, monetization ideas, and technical notes.
- `assets/` - source art, audio, references, and working files before import into Godot.
- `references/` - external research notes and general reference material.

Run smoke tests from the repository root:

```bash
./run_tests.sh
```
